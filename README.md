# 🚀 MyPortfolio V2

> Personal Portfolio Website - Version 2
> Designed & Developed by **Dung Nguyen**

---

## 📌 Giới thiệu

**MyPortfolio V2** là website portfolio cá nhân dùng để:

* Giới thiệu bản thân
* Trình bày kỹ năng (skills)
* Showcase các dự án đã thực hiện
* Làm CV online

---

## 🛠️ Công nghệ sử dụng

* **Frontend**: HTML, CSS, JavaScript
* **Animation**: GSAP (ScrollTrigger)
* **Responsive**: CSS + Media Query
* **Backend (nếu có)**: Node.js / Express
* **Database (nếu có)**: MSSQL / MySQL

---

## ✨ Tính năng nổi bật

* 🌌 Parallax scroll animation mượt (GSAP)
* 🌠 Hiệu ứng background (starfield / meteor)
* 📱 Responsive trên mobile, tablet, desktop
* ⚡ Tối ưu hiệu năng (60fps animation)
* 🎯 UI/UX hiện đại, tập trung trải nghiệm
